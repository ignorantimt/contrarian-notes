蒂尔是 PayPal 联合创始人、Facebook 首位外部投资人、Palantir 创始人，也是《从 0 到 1》（*Zero to One*）的作者。他在每个面试中都会问同一个问题：

> **在什么重要问题上，你的看法与大多数人不同？**

这个问题是本站的出发点，而蒂尔自己的答案，构成了一套相当完整的思想体系。

## 从 0 到 1：两种截然不同的进步

蒂尔区分了两种进步：

- **水平进步（1 → N）**：复制已经被验证的东西。全球化是典型——把发达国家的模式复制到发展中国家。
- **垂直进步（0 → 1）**：创造从未存在过的东西。科技是典型——从蒸汽机到计算机到互联网。

他的判断是：**如果只有全球化而没有科技创新，人类会陷入灾难**——几十亿人想过上发达国家的生活，靠现有技术只会导致资源竞争和环境崩溃。真正的未来取决于 0 到 1 的突破。

这个区分在今天依然锋利：大量所谓的"创业"其实是 1 到 N 的复制，把硅谷的模式搬到另一个城市、把美国的应用做一个本土版。复制能赚钱，但它不推动世界。

## 竞争是失败者的游戏

这是蒂尔最反直觉的主张：**不要竞争，要垄断。**

经济学教科书告诉我们，完全竞争是最理想的市场状态。蒂尔认为这完全搞反了：完全竞争意味着利润趋近于零，意味着你每天都在为生存挣扎，没有时间思考长期。航空公司服务着亿万乘客，却常年亏损；Google 一家独占搜索市场，利润厚到可以去探索登月项目。

他的方法论因此非常具体：

1. **从极小的市场开始垄断**——Facebook 只做哈佛校园，Amazon 只卖书，PayPal 只服务 eBay 上的高频卖家；
2. **垄断之后再向相邻市场扩展**；
3. **追求"后发优势"**（last mover advantage）——重要的不是第一个进场，而是在这个品类的终局里占据主导。

## 秘密：创新的矿藏

为什么还有人能做到 0 到 1？蒂尔的答案是：**世界上仍然存在未被发现的秘密**。

秘密分两种：关于自然的秘密（科学要解决的），和关于人的秘密（人们对自己隐瞒的事）。每一个伟大的公司都建立在一个秘密之上——Airbnb 的秘密是"陌生人愿意住进彼此的家里"，Uber 的秘密是"陌生人愿意上彼此的车"。这些在今天看来是常识，在当时是被所有人忽略的真相。

反过来，**一个不相信还有秘密的社会会停止创新**：如果一切都已被发现，剩下唯一理性的选择就是金融工程、法律咨询、在既有格局里内卷。蒂尔认为这正是当代西方社会的病灶之一。

## 幂次法则：世界不是平均分布的

我们习惯用正态分布理解世界，但蒂尔指出，风险投资——以及人生中很多重要的事——遵循**幂次法则**：少数几家公司、少数几个决定，贡献了绝大部分的回报。

推论相当激进：

- "多元化分散风险"在幂次世界里是幻觉，它保证你平庸；
- 你应该把最稀缺的资源（时间、注意力、资本）集中在少数真正可能指数级回报的事情上；
- 选择"做什么"远比"多努力"重要，因为在错误的方向上，努力的回报是线性的甚至是零。

## 明确的乐观主义

蒂尔把对未来的态度分成四个象限：

| | 乐观 | 悲观 |
|---|---|---|
| **明确** | 有计划的乐观（1950 年代的美国：登月、州际公路） | 有计划的悲观（蒂尔眼中的中国：储蓄、复制、追赶） |
| **不明确** | 无计划的乐观（当代美国：相信会更好，但不知道做什么，于是沉迷金融和多元化） | 无计划的悲观（欧洲：未来会变糟，只是不知道何时，于是提前享受） |

他推崇的是**明确的乐观主义**：相信未来会显著更好，并且给出具体的计划去实现它。创业公司的本质就是一小群人带着明确计划去改变世界——这与"不明确的乐观"者把未来交给概率和组合完全不同。

## 深层的人性解释：模仿欲望

蒂尔在斯坦福师从哲学家勒内·吉拉尔（René Girard），后者提出了**模仿欲望理论**：人的欲望不是自发产生的，而是通过模仿他人获得的——我们想要什么，往往是因为看到别人想要。

这解释了"竞争是失败者的游戏"的深层机制：**人们挤进同样的热门赛道，不是因为那些赛道机会最大，而是因为模仿让所有人欲望趋同**。金融、咨询、大厂、考研——越是精英聚集的地方，模仿性竞争越惨烈，而竞争者甚至没有停下来问过自己是否真的想要。

真正的创新者必须跳出这个循环：不是"比别人做得更好"，而是"做别人没有想过要做的事"。

## 批评与局限：蒂尔框架的边界

作为本站的第一篇分析，只复述不批评是不合格的。蒂尔的思想有几个明显的脆弱点：

**1. 幸存者偏差。** 垄断叙事是成功者的事后合理化。每一个讲"我先垄断小市场"故事的创始人背后，都有一万个同样起步但死掉的创始人。从结果倒推方法论，是商业写作的通病，蒂尔也不能免疫。

**2. "非共识且正确"的识别难题。** 蒂尔的框架告诉我们超额收益来自非共识且正确，但它没有告诉我们如何在事前区分"非共识且正确"和"非共识且愚蠢"。事后它们是显然的，事前它们长得一模一样。这是所有逆向思维的根本困难——**答案不在"反着来"，而在独立思考的质量**。

**3. 对竞争的否定被过度推广。** 在许多领域（体育、科学共同体内部、成熟的消费市场），竞争恰恰是进步的主要机制。蒂尔的反竞争论述最适用于"从 0 到 1 的技术创新"，把它当成普世人生哲学，可能会让人错过在竞争中磨炼的价值。

**4. 政治化倾向。** 蒂尔近年深度介入政治，其立场引发的争议时常盖过他的思想本身。读他的书，需要把"分析框架"和"政治议程"分开——前者值得认真对待，后者本文不作评价。

## 结语：这个博客的纪律

从这个思想体系里，本站取走的是三样东西，也是三条写作纪律：

1. **先问"共识是什么"**——说不清共识，就谈不上非共识。每篇分析从梳理主流观点开始。
2. **记录推理，而非结论**——结论会过时，推理链条可以被检验、被修正、被复用。
3. **写下可证伪的判断**——允许未来的自己回来打脸，打脸了就更新文章。这正是 git 的哲学，也是这个博客建在 GitHub 上的原因。

非共识不是目的，正确才是。非共识只是超额认知收益的入场券。
